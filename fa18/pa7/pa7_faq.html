w > v    is False
v == w    is False
v >= w   is True
w <= v   is True

    @memoized 
def foo(a,b,c): ...

    19. In the first case the decorated function would see 5,3,2 as positional arguments, and in the second it would see 5 as a positional argument, and c=2 and b=3 as keyword arguments.
