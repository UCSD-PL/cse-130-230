  1. Do we have to handle negative numbers in part 2?

    2. No. You can assume all big numbers are positive

  2. What is the behavior of bigAdd/mulByDigit/bigMul when multiplying by 0?

    3. # mulByDigit 0 [1;2;3];;

    # bigAdd [0] [0];;
    - : int list = []
    # bigMul [0] [1;2;3];;
    - : int list = []
    # bigMul [0;0;0] [1;2;3];;
