  3. I'm getting Indentation errors, but my indentation looks fine?

    4. The python interpreter differentiates between spaces and TAB-s. So even though two lines may look like they are indented the same width, if one is done with spaces, and the other with TAB-s python will complain. Make sure to be consistent and only use one or the other. One way to get your code to consistently use spaces, is to open it in vim and run the command ":retab"
